About

This project is about checking user input right away and using a simple queue. When someone fills out the form, it makes sure all the boxes are filled before letting them submit. After submitting, the info goes into a queue and shows up in a table. The “Next” button shows how a queue works by taking out the first thing that was added first (First In, First Out). This project helps me show that I get how to check forms
